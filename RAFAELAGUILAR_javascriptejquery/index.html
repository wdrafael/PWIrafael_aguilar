Java Script

De Java, só o nome...

Java Script é uma linguagem de programação web que torna o site mais interativo.

Um mapa, um horário que fica atualizando, botões e efeitos são frutos de um script Java Script.

Ficou claro como o Java Script complementa o HTML e o CSS no desenvolvimento de páginas web.

Um recurso muito comum e bacana de Java Script é a possibilidade de bloquear o botão direito do mouse. Pode tentar, você não vai conseguir!

Ao tentar clicar com o botão direito, apareceu uma mensagem, correto? Este é outro exemplo de alerta, feito em Java Script. Para visualizar o efeito novamente, basta CLICAR AQUI!.

Se quiser copiar os códigos, fique à vontade:

Para bloquear o botão direito do mouse, utilize o script:


<script language="JavaScript">
function click(){
if (event.button== 2) {
alert("BOTÃO DIREITO DO MOUSE BLOQUEADO!")}
}
</script>

Agora, se quiser saber como colocar o alerta na página, segue o código:

<script>
function alert()}
alert('Exemplo de alerta criado em Java Script');
}
 </script>
