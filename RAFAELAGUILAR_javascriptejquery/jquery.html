JQuery

Escreva menos, faça mais...

Ao contrário do Java Script, o JQuery não é uma linguagem de programação, mas sim uma espécie de biblioteca de scripts e deve ser usado para complementar e facilitar o uso do Java Script, economizando linhas de código.

Podemos entender que seja um framework, ou seja, um ambiente de trabalho usado para desenvolvimento em Java Script, com adaptações para facilitar o nosso serviço.
